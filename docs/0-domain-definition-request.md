너는 최고의 비즈니스 도메인 전문가야. "Python Judge"라는 이름의 python을 이용한 알고리즘 코딩 테스트를 수행할 수 있는 애플리케이션을 만들고자 해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘.

# 문제 정의 :

- 학생에게 python 알고리즘 문제를 ppt로 보여주고, 학생에게 \*.py 파일을 제출받으면 해당 파일을 일일히 실행하며 옳게 작성했는지 확인해야 하므로 번거롭고 오류 가능성이 있다.
- 학생의 정오답에 따른 점수 정보를 하나하나 체크해야 해서 번거롭다.
- 학생이 실시간으로 자신이 작성한 코드의 정답 여부를 알 수 없으므로 문제 해결 시간이 늘어난다.

# Python Judge 앱의 주요 기능 :

- 관리자(선생님) 계정은 미리 존재하고 있으며, 별도의 관리자 페이지를 가지고 있다.
- 학생은 회원가입을 통해 등록 가능하며, 학생은 풀어야 할 문제들을 카테고리별로 확인할 수 있다.
- 학생은 회원가입을 위해 ID, 이름, 비밀번호를 필수적으로 입력한다. 군번은 수집하지 않는다.
- 알고리즘 문제는 구현해야 할 프로그램의 기능, 입/출력의 범위, 예제 입/출력 등을 제공한다.
- 알고리즘 문제는 python을 통해 해결할 수 있도록 한다.
- 알고리즘 문제의 범위는 변수, 연산자, 자료형, 조건문, 문자열, 리스트, 튜플, 반복문, 딕셔너리, 집합, 함수, 파일 입/출력, 클래스, 예외 처리, 모듈과 패키지 순으로 진행되는 교육 과정에서 순차적으로 출제됨을 원칙으로 한다.
- 조건문, 반복문, 딕셔너리, 파일 입/출력, 예외 처리 이후에 각각 풀어야 할 문제 세트를 만든다.
- 학생에게는 일부의 테스트 케이스만 예시로 주어지고, 채점 시에는 일부의 테스트 케이스를 부분집합으로 하는 큰 테스트 케이스 집합을 모두 통과했는지 여부에 따라 정오답을 채점한다.
- 학생은 어떤 테스트 케이스에서 틀렸는지 상세하게 확인할 수 없다.
- 관리자는 교육 세션을 설정하여, 특정 시점 기준으로 문제해결 내역과 스코어보드를 초기화할 수 있다.
- 관리자는 이 python judge 애플리케이션을 통해 학생들의 학업 성취도 평가를 할 수 있다.
